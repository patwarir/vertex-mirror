A datatype is the type of a piece of data, or the absence of the said piece.
They are read from the stream buffer.

Hex Index	0x00
Title		Void
Alias		void
Default		N/A
Byte(s)		N/A
Summary		Represents the absence of a value.
			NOTE: Void can only be used on function return types; no values can be of type void.

Hex Index	0x01
Title		Boolean
Alias		bl
Default		false
Byte(s)		1
Summary		Represents a true or false value.

Hex Index	0x02
Title		Integer
Alias		int
Default		0
Byte(s)		8
Summary		Represents a 64-bit signed, non-decimal number.

Hex Index	0x03
Title		Float
Alias		fl
Default		0.0
Byte(s)		8
Summary		Represents a 64-bit signed, decimal (floating-point) number.

Hex Index	0x04
Title		String
Alias		str
Default		""
Byte(s)		N/A
Summary		Represents a variable list of characters.
			NOTE: A string cannot be null.
