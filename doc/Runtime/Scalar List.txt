A scalar list represents a variable-length collection of Scalars.

Code
public Constant { get; set; } -> bool
	This boolean represents if the list is constant, as in it cannot change after creation.
	It defaults to false.

public ScalarList(bool [false]) -> void
	This constructor creates the list and by default sets the constant value to false.

public Add(byte, Datatype, object) -> void
	This method adds a Scalar to the list at a specific index.

public Change(byte, object) -> void ~ InvalidOperationException
	This method changes a Scalar in the list at a specific position.
	It throws an InvalidOperationException if the list is declared to be constant.

public GetValue(byte) -> object
	This method returns the value of the Scalar at the specific index.

public GetDatatypes() -> IEnumerable<Datatype>
	This method returns an IEnumerable of all the datatypes of the Scalars in the list.
