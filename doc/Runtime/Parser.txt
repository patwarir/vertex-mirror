A parser converts a text file to a list of functions with their properties.

Code
public Functions { get; } -> IList<Function>
	This is the output of the Parser, the functions.

public Parser(string) -> void
	Creates a new Parser with the specified code string.

private TrimTabs(string) -> string
	Trims the tabs from the beginning of a string.

private GetDatatype(string) -> Datatype
	Gets the datatype of a specific string.

private GetDefaultValue(Datatype) -> object
	Gets the default value of a specific datatype.

private GetStringLiteral(string) -> string
	Gets a string without outside quotes.

public Run() -> void
	Runs the Parser over every line in the given code.
