The interpreter takes the stream buffer and executes the instructions that it contains.

Code
public Interpreter(IEnumerable<Function>) -> void
	Creates an interpreter with the specified functions.

private GetLabel(Function, string) -> Label
	This method finds a label with the specified name in the specified function.

private GetFunction(string, Datatype, IEnumerable<Datatype>) -> Function
	This method finds a function with a specified name, return datatype and parameter types.

private Check(Datatype, object) -> object
	This method checks an object to be consistent with a specified datatype.

private Arithmetic(OperationCode, object, object) -> object
	This method performs the specified arithmetic operation with the given values.

private Conditional(OperationCode, object, object) -> bool
	This method returns whether the conditional type (equals, etc.) is true with the specified values.

private RunFunction(Function) -> void
	This method runs the specific function and performs operations based on its stream buffer.
	Each operation code instructs the virtual machine that executes the code.

public Run() -> void
	This method calls the RunFunction method with the "main" function that has no parameters and returns void.
